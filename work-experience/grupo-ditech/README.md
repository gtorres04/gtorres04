## English

**Role:** Java Architect
**Period:** October 2019 - April 2021 (1 year 7 months)
**Location:** Madrid, Spain

---

Acted as Java Architect and Technical Lead at Grupo Ditech for Evo Banco and Bankinter, ensuring quality, architectural coherence, and governance of critical platforms.

### Projects

#### Evo Banco

The main challenge was integrating multiple third-party APIs into the corporate API Gateway, imposing RESTful standards, normalizing inconsistent contracts, and correcting bad practices without degrading security, compliance, or performance. In parallel, I led the design and development of a mortgage simulation microfrontend, aligning architecture and business processes to maintain functional consistency, low latency, and traceability.

#### Bankinter

Led the construction of a configuration management and versioning platform for microservices, capable of handling sensitive parameters without compromising environment stability, integrating Consul as the single source of configuration and reducing errors from misalignments between services and versions.

### Solution

Solutions were implemented with Java 8 and Spring Boot on the backends, ReactJS and Angular 9 on the frontends, and a corporate API Gateway deployed on Azure Cloud for Evo Banco. Consul was used as a centralized configuration repository, GitHub for version control, Jenkins for CI/CD, Docker and Azure App Services for containerization and deployments, and Azure API Management for API governance. Work was organized under Scrum, with continuous integration, code reviews, automated testing, and practices such as pair programming and close coordination with architecture, QA, and business teams.

The result was a more robust and standardized integration ecosystem, reliable configuration management, and a modern mortgage simulator that improved customer experience and reduced operational incidents at both banks.

### Tech Stack

- **Languages:** Java 8
- **Frameworks:** Spring Boot
- **Frontend:** ReactJS, Angular 9, microfrontends
- **Configuration:** Consul
- **Cloud:** Azure (Azure App Services, Azure API Management)
- **CI/CD:** Jenkins, Docker
- **Methodology:** Scrum, pair programming, code reviews
