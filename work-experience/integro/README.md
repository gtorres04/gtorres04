## English

**Role:** Senior Java Developer
**Period:** September 2017 - November 2017 (3 months)
**Location:** Medellín, Colombia

---

Developed a digital wallet for the US market at Integ.ro, aimed at enabling payments and automatic micro-savings through an intelligent transaction rounding mechanism.

The main technical challenge was implementing a payment module capable of rounding amounts to two decimal places and automatically transferring the difference to a profitable savings pocket, ensuring transactional integrity, financial accuracy, and full consistency between debit and credit movements.

### Business Challenges

The complexity lay in ensuring a reliable, transparent, and traceable payment flow for the end user, complying with regulatory requirements and maintaining full visibility of the lifecycle of each operation.

### Solution

The solution was developed with microservices in Java 8 and Spring Boot on MySQL, applying unit tests, integration tests, and resilience-oriented design principles. On the frontend I worked with Angular and React Native, building modular and reusable components. The project used Docker for containerization, cloud deployments, CI/CD with GitHub, Sonar, Jira, and automated pipelines; TDD practices, functional tests, and manual validations were also applied to reinforce quality.

Work was organized under Scrum, managing tasks through user stories, holding periodic retrospectives, and applying continuous improvement across all development cycle processes.

The result was a stable, secure payment and micro-savings module fully aligned with the financial product's operational requirements, improving user experience and strengthening the reliability of the wallet's digital ecosystem.

### Tech Stack

- **Languages:** Java 8
- **Frameworks:** Spring Boot (microservices)
- **Frontend:** Angular, React Native
- **Data:** MySQL
- **CI/CD:** GitHub, Sonar, Jira, Docker
- **Methodology:** Scrum, TDD

## Español

**Cargo:** Desarrollador Senior Java
**Período:** septiembre de 2017 - noviembre de 2017 (3 meses)
**Ubicación:** Medellín, Colombia

---

Desarrollé una billetera digital para el mercado estadounidense en Integ.ro, orientada a habilitar pagos y micro-ahorros automáticos mediante un mecanismo de redondeo inteligente de transacciones.

El principal reto técnico consistió en implementar un módulo de pago capaz de redondear importes a dos decimales y transferir automáticamente la diferencia a un bolsillo de ahorro rentable, garantizando integridad transaccional, exactitud financiera y plena consistencia entre movimientos de débito y abono.

### Retos de negocio

La complejidad radicó en asegurar un flujo de pago confiable, transparente y trazable para el usuario final, cumpliendo requisitos regulatorios y manteniendo visibilidad completa del ciclo de vida de cada operación.

### Solución

La solución se desarrolló con microservicios en Java 8 y Spring Boot sobre MySQL, aplicando pruebas unitarias, de integración y principios de diseño orientados a resiliencia. En frontend trabajé con Angular y React Native, construyendo componentes modulares y reutilizables. El proyecto utilizó Docker para contenedorización, despliegues en cloud, CI/CD con GitHub, Sonar, Jira y pipelines automatizados; además, se aplicaron prácticas TDD, pruebas funcionales y validaciones manuales para reforzar la calidad.

La organización del trabajo siguió Scrum, gestionando las tareas mediante historias de usuario, realizando retrospectivas periódicas y aplicando mejora continua en todos los procesos del ciclo de desarrollo.

El resultado fue un módulo de pagos y micro-ahorros estable, seguro y totalmente alineado con los requisitos operativos del producto financiero, mejorando la experiencia del usuario y fortaleciendo la confiabilidad del ecosistema digital de la billetera.

### Stack tecnológico

- **Lenguajes:** Java 8
- **Frameworks:** Spring Boot (microservicios)
- **Frontend:** Angular, React Native
- **Datos:** MySQL
- **CI/CD:** GitHub, Sonar, Jira, Docker
- **Metodología:** Scrum, TDD
