## English

**Role:** Software Engineering Analyst
**Period:** November 2017 - October 2018 (1 year)
**Location:** Antioquia, Colombia
**Client:** Bancolombia

---

Designed and developed a documentary consultation REST API for Bancolombia that centralizes and uniformly exposes scanned customer documents integrated with Oracle WebCenter Content (WCC).

The main challenge was thoroughly understanding the proprietary API of the document manager, mapping its internal operations, and transforming heterogeneous models and structures into standardized REST contracts aligned with the bank's architectural standards. I also had to resolve format differences, validate metadata, and ensure consistent behavior under high-volume requests from multiple internal channels.

### Business Challenges

The challenge focused on guaranteeing stable response times, complying with strict security and audit policies on sensitive information, and providing exhaustive traceability of each query. This involved designing control, logging, and validation mechanisms that allowed operations to be tracked end-to-end, maintaining integrity, confidentiality, and regulatory compliance throughout the flow.

### Solution

The solution was implemented with Java 6 and JEE, exposing REST and SOAP services on WebSphere Application Server, integrating with the document manager via Oracle WCC-specific APIs. The project was accompanied by automated and functional tests executed with VSTS, version control, and corporate environment deployments managed iteratively, in direct coordination with architecture, security, and the bank's functional teams.

The result was a single, standardized, and reliable point for customer document consultation that reduced access times, decreased operational errors from individual integrations, and facilitated service reuse in new processes, channels, and Bancolombia's internal applications.

### Tech Stack

- **Languages:** Java 6, JEE
- **Services:** REST, SOAP
- **Server:** WebSphere Application Server
- **Document management:** Oracle WebCenter Content (WCC)
- **Testing:** VSTS

## Español

**Cargo:** Analista Ingeniería de Software
**Período:** noviembre de 2017 - octubre de 2018 (1 año)
**Ubicación:** Antioquia, Colombia
**Cliente:** Bancolombia

---

Diseñé y desarrollé para Bancolombia una API REST de consulta documental que centraliza y expone de forma unificada los documentos escaneados de clientes integrados con Oracle WebCenter Content (WCC).

El reto principal fue comprender a profundidad el API propietario del gestor documental, mapear sus operaciones internas y transformar modelos y estructuras heterogéneas en contratos REST estandarizados y alineados con las normas arquitectónicas del banco. También debí resolver diferencias de formatos, validar metadatos y asegurar un comportamiento consistente ante solicitudes de alto volumen provenientes de múltiples canales internos.

### Retos de negocio

El desafío se centró en garantizar tiempos de respuesta estables, cumplir estrictas políticas de seguridad y auditoría sobre información sensible y ofrecer trazabilidad exhaustiva de cada consulta realizada. Esto implicó diseñar mecanismos de control, registro y validación que permitieran rastrear operaciones de inicio a fin, manteniendo integridad, confidencialidad y cumplimiento normativo en todo el flujo.

### Solución

La solución se implementó con Java 6 y JEE, exponiendo servicios REST y SOAP sobre WebSphere Application Server, integrándose con el gestor documental mediante las APIs específicas de Oracle WCC. El proyecto se acompañó de pruebas automatizadas y funcionales ejecutadas con VSTS, control de versiones y despliegues en entornos corporativos gestionados de manera iterativa, en coordinación directa con arquitectura, seguridad y los equipos funcionales del banco.

El resultado fue un punto único, estandarizado y confiable para la consulta documental de clientes, que redujo tiempos de acceso, disminuyó errores operativos derivados de integraciones individuales y facilitó la reutilización del servicio en nuevos procesos, canales y aplicaciones internas de Bancolombia.

### Stack tecnológico

- **Lenguajes:** Java 6, JEE
- **Servicios:** REST, SOAP
- **Servidor:** WebSphere Application Server
- **Documental:** Oracle WebCenter Content (WCC)
- **Testing:** VSTS
